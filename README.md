# CSE560_Assignment5_BayesianNetworks
CSE560 Advanced Artificial Intelligence Course - Assignment5 BayesianNetworks

You should run main.py file to run the program.
In order to configure the topology of the Bayesian Network(the connections between the nodes), you should use bn.txt file.
The question you want to ask should be written to Question.txt file.

For example if you want to see the probability of I being True, H being False, L being False, P being True and E being False:
you should use IT,HF,LF,PT,EF in the Question.txt file.
